# 🏋️‍♂️ FitBuddy – AI Powered Fitness Tracking App

FitBuddy is a full-stack **MERN-based fitness tracking application** that helps users monitor their progress, log workouts, follow structured plans, and get guidance from an AI-powered fitness assistant.

## 🚀 Features

### 📊 Track Progress
- Monitor workouts and fitness journey over time

### 📝 Workout Logging
- Log daily exercises and maintain consistency

### 🏋️ Custom Workout Plans
- Create personalized workout routines

### 📦 Pre-built Workout Plans
- Push Pull Legs (PPL)
- Fat Loss Programs  
- Customized based on:
    - Gender (Male / Female)
    - Experience Level (Beginner / Intermediate / Advanced)
### 🤖 AI Fitness Assistant
- Powered by OpenRouter (NVIDIA Nemotron model)
- Ask questions about:
  - Diet
  - Workouts
  - Fitness goals
- Acts like your personal fitness buddy

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### AI Integration
- OpenRouter API (NVIDIA Nemotron Model)
